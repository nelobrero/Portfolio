Servicita

A smart mobile application that connects service providers (like plumbers, electricians, tutors,
etc.) with people who need their help. Using AI recommendations, it suggests the best available
professionals based on location, availability, and user preferences. It also includes real-time
navigation and chat features to streamline communication and service delivery.

Loterya

A network-based multiplayer game inspired by the traditional Mexican game “Lotería,"
similar to Bingo. This project was developed to apply the fundamentals of socket
programming and peer-to-peer communication.Players connect to a shared session, receive
randomized cards, and play together in real-time.

Honey OS

An experimental operating system built from scratch with a focus on voice-command functionality.
Honey OS allows users to navigate, launch apps, and execute commands through voice alone.
This project explores low-level OS development, custom shell design, and voice recognition
integration to create a more intuitive human-computer interaction.

Woredrobe

A mobile application designed to help users organize their wardrobe and plan outfits.
Users can catalog their clothing items by uploading photos and adding details like color,
style, and occasion. The app offers outfit suggestions based on weather, trends, and user
preferences, making it easier to decide what to wear each day.
